## 📐 布局設計

### 整體結構
```
┌─────────────────────────────────────────┐
│          Header（保留）                   │
├──────────┬──────────────────────────────┤
│          │                              │
│  左側     │      右側內容區                │
│  菜單欄   │      (Router View)           │
│          │                              │
│  [獨立    │      [獨立滾動條]             │
│   滾動條] │                              │
│          │                              │
├──────────┴──────────────────────────────┤
│          Footer（保留）                   │
└─────────────────────────────────────────┘
```

### 左側菜單區塊
- **寬度**: 固定 250px（可調整）
- **滾動**: 獨立垂直滾動條
- **背景**: 深色主題（#2c3e50 或類似）
- **結構**: 二級菜單（父項目 → 子項目）

### 右側內容區塊
- **寬度**: 自適應剩餘空間
- **滾動**: 獨立垂直滾動條
- **背景**: 淺色背景（#f5f5f5）
- **內容**: Router View（動態切換）

---

## 🗂️ 菜單結構範例

### 建議的二級菜單架構

```
📊 儀表板
   └─ 總覽
   └─ 統計圖表

🔧 系統管理
   └─ 系統資訊
   └─ 系統設定

📖 關於
   └─ 應用資訊
   └─ 版本歷史
```

### 互動行為
1. **點擊父項目** → 展開/收合子項目
2. **點擊子項目** → 切換右側 Router 內容
3. **視覺回饋** → 當前選中項目高亮顯示
4. **展開狀態** → 使用圖標指示（> 或 ▼）
