# REQ-010: 應用設定檔保存機制與資料庫連線設定

> 需求編號：REQ-010  
> 建立日期：2026-06-03  
> 狀態：📝 需求階段  
> 優先級：P1

---

## 📋 需求概述

建立應用程式外部設定檔機制，使用 JSON 格式保存應用設定，並在設定頁面新增資料庫連線配置功能。

---

## 🎯 需求目標

### 核心功能

1. **外部設定檔機制**
   - 使用 JSON 格式保存應用設定
   - 設定檔位置：應用程式資料目錄
   - 支援讀取、寫入、更新設定

2. **資料庫連線設定**
   - 支援資料庫類型：MS SQL、MySQL
   - 連線參數配置：
     - 資料庫種類（下拉選單）
     - Host（主機位址）
     - Username（使用者名稱）
     - Password（密碼）
   - 設定保存至 JSON 檔案

### 技術需求

- ✅ JSON 設定檔讀寫
- ✅ 設定表單驗證
- ✅ 設定變更即時保存
- ✅ Electron 檔案系統整合


---

## 📊 功能詳細說明

### 1. 設定檔結構

```json
{
  "version": "1.0.0",
  "database": {
    "type": "mssql",  // "mssql" | "mysql"
    "host": "localhost",
    "username": "admin",
    "password": "your_password"
  },
  "language": "zh-TW",
  "lastModified": "2026-06-03T12:00:00.000Z"
}
```

### 2. 設定頁面 UI

**新增區塊：資料庫連線設定**
- 資料庫種類選擇（單選）
  - MS SQL
  - MySQL
- Host 輸入欄位（文字輸入）
- Username 輸入欄位（文字輸入）
- Password 輸入欄位（密碼輸入，顯示/隱藏切換）
- 測試連線按鈕
- 儲存設定按鈕
